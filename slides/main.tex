\begin{frame}{Бинаризация изображений}
Бинаризация изображения - важный этап, предшествующий детекции строк.

\begin{block}{Проблема}
Стандартные методы бинаризации (Оцу, глобальный порог) плохо работают при переменном освещении.
\end{block}

\begin{block}{Предлагаемый подход: автокодировщик}
Используем свёрточный автокодировщик для классификации каждого пикселя на два класса: текст/фон.
\end{block}

\begin{block}{Результат}
\begin{enumerate}[1)]
    \item построена архитектура U‑Net,
    \item повышение качества бинаризации на сложных изображениях.
\end{enumerate}
\end{block}
\end{frame}
